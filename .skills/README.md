# Skills Directory

This directory contains specialized Claude skills for enhanced workspace functionality.

## Structure

```
.skills/
├── README.md              # This file
├── todo/                   # TODO-related skills (future)
│   └── SKILL.md           # Advanced task management behaviors
└── [future-skills]/        # Other specialized capabilities
    └── SKILL.md
```

## What Are Skills?

Skills are specialized Claude behaviors that provide:
- **Context-specific expertise** for complex tasks
- **Enhanced processing** beyond basic commands  
- **Domain knowledge** for specialized workflows
- **Custom interactions** for advanced use cases

## Creating Skills

1. **Create directory**: `.skills/skill-name/`
2. **Add skill file**: `SKILL.md` with specialized instructions
3. **Reference in instructions.md**: Include in main context
4. **Test and iterate**: Refine based on usage

## Skill Template

```markdown
# [Skill Name] Skill

Specialized Claude behavior for [specific domain/task].

## Purpose
[What this skill provides]

## Activation
[When this skill should be used]

## Specialized Knowledge
[Domain expertise, patterns, best practices]

## Example Interactions
[How this skill enhances normal Claude behavior]
```
